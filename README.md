## 📌 Overview

**Suriyawan Saffari** is a complete, production-ready e-commerce platform with **4 interconnected portals** and **full barcode system**.

| Portal | Features | Users |
|--------|----------|-------|
| 🛍️ **Customer Portal** | 100+ features | Buyers |
| 🏪 **Seller Portal** | 100+ features | Merchants |
| 🚚 **Logistics Portal** | 150+ features | Hub Managers + Riders |
| 👑 **Owner Portal** | 110+ features | Admin |

### 🎯 Key Highlights

- ✅ **COD Only** - No online payment, No OTP, No UPI for payment
- ✅ **Cancel Code System** - 6-digit code, 48hr expiry, only customer has it
- ✅ **Open Box Delivery** - Customer opens product in front of rider
- ✅ **Same Time Return/Exchange** - Instant pickup and replacement
- ✅ **20 IDs System** - AUTO + MANUAL ID generation
- ✅ **Barcode System** - PROD ID, BOOK ID, AWB, Cancel Code, Invoice barcodes
- ✅ **Invoice + Label Print** - GST invoice with barcode, shipping labels
- ✅ **Runsheet System** - AI-optimized delivery routes
- ✅ **40+ Product Tags** - Hot, Premium, Viral, Trending, etc.
- ✅ **Real-time Tracking** - Live rider location on map

## 🔖 BARCODE SYSTEM

### Barcode Types

| Barcode Type | Format | Used For |
|--------------|--------|----------|
| PROD ID | PROD202605080001 | Product identification |
| BOOK ID | BOOK202605080001 | Order tracking |
| AWB | SS202605080001 (12-digit) | Shipment tracking |
| Cancel Code | 847291 (6-digit) | Order cancellation |
| Invoice | INV202605080001 | GST bill verification |
| RETURN ID | RET202605080001 | Return tracking |
| EXCH ID | EXCH202605080001 | Exchange tracking |
